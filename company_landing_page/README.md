# Veritas Consulting — Company Landing Page

A professional, responsive company landing page built with semantic HTML5 and CSS3.

## Project Structure

```
company_landing_page/
├── index.html    ← All page sections + embedded JS
├── style.css     ← Full stylesheet
└── README.md     ← This file
```

## Sections Covered

| Section | Task |
|---------|------|
| Header / Hero | Task 2 |
| About / Introduction | Task 2 |
| Services (6 cards, grid layout) | Tasks 2 & 4 |
| Testimonials (3 clients) | Tasks 2 & 5 |
| Contact (info + form) | Tasks 2 & 6 Bonus |
| Footer | Task 2 |

## How to Run

**Option A — Live Server (recommended)**
1. Open `company_landing_page/` in VS Code
2. Install the **Live Server** extension (Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**

**Option B — Direct open**
- Double-click `index.html` in any modern browser

## Features

- Fixed navbar that turns solid on scroll
- Animated hero with scrolling ticker strip
- CSS scroll-reveal animations on all sections
- Responsive grid layout (3-col → 2-col → 1-col)
- Contact form with validation + success confirmation
- CSS variables for consistent theming
- Google Fonts: Playfair Display + Barlow
